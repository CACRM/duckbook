Line 4 - IRA Distributions

1099-R Form ‚Äì Distributions from Retirement Accounts

  • Form 1099-R is used for two somewhat similar things: distributions from retirement accounts, and pension and annuity payments. Distributions are covered in this section; pensions and annuities (Line 5) are covered in a later section, page 38.
    • Generally, if a Form 1099-R is for a retirement account, rather than a pension, then the checkbox ‚ÄúIRA/SEP/SIMPLE‚Äù (on the paper form, the checkbox is near box 7), will be checked.
      • TaxSlayer only puts income from the form onto Form 1040 Line 4 if the checkbox is checked in TaxSlayer.
  • Box 2a:
    • When an amount is entered in box 1, TaxSlayer automatically uses that to fill box 2a, with a note that the amount in box 2a can be changed.
      • If the amount that TaxSlayer automatically enters in box 2a matches the box 2a amount on the paper Form 1099-R, then continue down the form.
    • If the amount that TaxSlayer automatically enters in box 2a does not match box 2a on the paper form:
      • If the paper Form 1099-R has a non-zero amount in box 2a, enter it in TaxSlayer in box 2a.
      • If box 2a is a zero (not just blank), and box 2b is not checked (‚ÄúTaxable amount not determined‚Äù), then start by presuming that none of the distribution is taxable. This needs to be confirmed. Look at box 5, employee contributions. If the box 5 amount is the same as box 1, then none of the distribution is taxable, so box 2a in TaxSlayer should be changed to zero. Else, discuss with the taxpayer. 
      • If 2a is blank (this is rare), box 2b is normally checked (‚ÄúTaxable amount not determined‚Äù). To determine taxability, look at prior returns and discuss with the taxpayer. 
      • If there‚Äôs not already a TaxSlayer note regarding distributions of this type, add one to help counselors in future years (page 8).
  • Checkbox below box 2a
    • Some distributions do not affect the calculations for the Retirement Savings Contribution Credit (page 91) ‚Äì distributions that are rollovers, or are related to Roth IRAs, or otherwise should not be included ‚Äì see the list on NTTC 4012 page G-21.
      • For such distributions, check the ‚ÄúDoes not qualify for Form 8880‚Äù box to keep these distributions from affecting Form 8880 calculations. 
  • Box 7 code: Take care in entering the code on the Form 1099-R.
    • The IRS Instructions for Form 1099-R has a ‚ÄúGuide to Distribution Codes‚Äù which is more detailed than the table on NTTC 4012 pages D-63 to 64.
    • If multiple codes are shown (such as ‚ÄúG4‚Äù), enter both codes in TaxSlayer.
      • For issues of scope, check each code separately. 
    • Code 6 (tax-free Section 1035 exchange), code Q (Roth IRA qualified distribution), and code W (payment for a LTC contract) are not taxable; do not report on the return.
      • For forms with code W, do not claim the amount in box 1 as a medical expense on Schedule A, or as medical insurance costs for SEHI deduction, because the distribution is tax-free (box 2a is zero).
    • Code Q is for a Roth IRA qualified distribution ‚Äì enter 0 in Box 2a in TaxSlayer.
    • Code D (annuity payments from nonqualified annuities) ‚Äì only the gain is taxable.  If box 2a on Form 1099-R is blank, call the issuer (payer) to get the taxable amount, if any.
    • Code G and code H are direct rollovers ‚Äì see the section 1099-R Form ‚Äì Transfer or Rollover, page 37.
    • Code J (early distribution from a Roth IRA) ‚Äì follow the instructions on  NTTC 4012 page D-64 to find whether the Form 1099-R is in scope, and if so, how to enter in TaxSlayer.
    • Code T is for a distribution from a Roth IRA if the participant has reached age 59¬Ω but the 5-year holding period may not have been met. Use the flow chart on NTTC 4012 page D-64 to determine taxability (if any of the distribution is taxable, the return is out-of-scope).
      • Note: If, after completing the TaxSlayer page for a 1099-R with code T, TaxSlayer displays the Form 8606 ‚Äì Nondeductible IRAs page, the return is out-of-scope, because Part III needs to be completed, which is out-of-scope.
    • Code Y:  designates a QCD; see section 1099-R Form ‚Äì Qualified Charitable Distribution, page 38.
    • Code 1 or Code S:  after all information is entered in TaxSlayer, and ‚ÄúContinue‚Äù or ‚ÄúSave & Enter Another‚Äù is clicked, the Form 1099-R Distribution Penalty page appears. 
      • For ‚ÄúWhat type of plan did you receive this distribution from?‚Äù, check ‚ÄúRetirement Plan.‚Äù
      • For the second question, determine if there is an exception to the additional tax for early distribution (before age 59 ¬Ω).
        • See the section 1099-R Form ‚Äì Exceptions to the Additional Tax on Early Distributions, page 36, for determining if the taxpayer can claim an exception to the penalty.
        • If there is an exception, check the box. [Ignore the sentence ‚ÄúNot sure if your withdrawal qualifies to be exempt from the penalty?‚Äù] 
        • Entering the exception code, and the exception amount, must be done separately, on the Form 5329 page, as discussed in the next section. It is best to do that immediately after finishing the Form 1099-R Distribution Penalty page.
      • Click ‚ÄúContinue‚Äù to finish the penalty page.
  • Box 7 checkbox: If the ‚ÄúIRA/SEP/SIMPLE‚Äù box is checked:
    • Confirm with the taxpayer whether it‚Äôs a Roth conversion.
    • A conversion from a traditional IRA to a Roth IRA requires completing Form 8606 Part II. 
    • Distribution from a converted Roth IRA makes the return out-of-scope if any portion of the distribution is or could be taxable.
      • Note that the 5-year rule (penalty for withdrawal from a Roth IRA within five years of opening that account) applies to each conversion.
  • Box 9b
    • If the taxpayer cashed in an annuity, they‚Äôll have a Form 1099-R with the amount they received.
    • If the investment (in box 9b; may also be in box 5) is greater than the amount shown in box 1, the net loss can be claimed as a miscellaneous itemized deduction ‚Äì see Schedule A ‚Äì Other Itemized Deductions (Line 16), page 83.

Distributions from an IRA Where Over-Age-70¬Ω Contributions Have Been Made

  • On the federal return, a distribution from an IRA where over-age-70¬Ω deductible contributions have been made is treated no differently from a distribution from an IRA with no such contributions.
  • CA policy, new for TY2025: CA now conforms to federal rules, beginning Jan. 1, 2025.  For 2024 and earlier years, CA does not conform to federal rules regarding such contributions (they are not a reduction of income, as discussed on page 70), and so should not treat the related distributions as income.
    • However, it‚Äôs difficult-to-impossible to determine the portion of the withdrawal involving money that California previously did not allow as a reduction of CA income. Even if adequate taxpayer or IRA custodial records are available, the allocation of the distribution almost certainly involves assumptions, complicated by trying to account for earnings or losses inside the IRA.
    • Thus, it is CA policy for prior year returns with IRA distributions from an account where contributions were made after age 70¬Ω, there is no CA adjustment (no reduction of income): 100% of such distributions are taxable on the CA return.
      • If the taxpayer wants to reduce the CA taxability of that IRA distribution, the return is out-of-scope.

Required Minimum Distributions (RMDs) 

  • If the taxpayer (or the spouse if MFJ) with a traditional IRA was at least age 73 by Dec. 31 of the tax year, then they were normally required to begin taking distributions from that IRA. 
    • If the taxpayer or spouse reached that age, and there are no Form 1099-Rs in the documents provided by the taxpayer that involve distributions, discuss whether the taxpayer or spouse has any traditional IRAs, and if so, their RMD options.
    • For tax year 2024, there is an exception for certain inherited IRAs where the original owner died in 2020, 2021, 2022, or 2023:  if the owner had reached their required beginning date (typically April 1 after they turn 73), no RMD was required for 2024, and thus no penalty for not taking it.
    • If there is any question as to whether the taxpayer did in fact take their RMD, see section 5329 Part IX ‚Äì RMD Not Taken, page 96, which includes a discussion of how to determine RMD.

1099-R Form ‚Äì Exceptions to the Additional Tax on Early Distributions 

  • TaxSlayer automatically assesses an additional tax of 10% for an early distribution (prior to age 59¬Ω) from an IRA; early distributions are those on Form 1099-Rs with code 1 or code S in Box 7. 
    • Don‚Äôt use the taxpayer‚Äôs age as of December 31 of the tax year to determine whether a penalty applies; use the taxpayer‚Äôs actual date of birth, and the actual date of withdrawal to find the age at withdrawal.
    • Note for CA returns: CA also assesses a 2.5% tax for early withdrawals; this shows on CA Form 3805P and on line 63 of Form 540. (California conforms to all federal exceptions to the penalty.)
  • Exceptions to the additional tax are listed in Pub 4012 page H-7.
    • For additional exceptions that apply to annuities, see Pub 575, Pension and Annuity Income.
    • Note that exceptions 07 through 09 are only for IRA distributions, not distributions from other types of qualified retirement plans.
    • For exception 08, qualified higher education expenses can include room and board. See Pub 590-B, Distributions from Individual Retirement Arrangements, for details on that and other exceptions. 
    • Beginning in 2024, there is a new exception for owners of any eligible retirement plan for withdrawals to cover eligible emergency expenses, but maximum is $1,000 and there are caveats.
        • Unless repaid, taxpayer can‚Äôt take a similar emergency exception for the next three tax years.  Add a TaxSlayer note (page 8) to say "For TY24, taxpayer used $XXXX emergency exception to early withdrawal penalty."
        • Any subsequent IRA contribution, in the same tax year or the next three calendar years is automatically considered a repayment; the taxpayer doesn‚Äôt have to designate it as repayment.
    • Effective for distributions after 12/29/2022, there are exceptions for distributions due to terminal illness, and for private-sector firefighters, public-safety officers (PSO) with 25 years of service, and state and local gov‚Äôt corrections officers.  Note:  This definition of PSO is broader than the one for 1099-R Form ‚Äì Public Safety Officer (PSO) Exceptions on page 41. See Pub 4012 page H-7 and Retirement Topic - Exceptions to Tax on Early Distributions at irs.gov. 
    • Funds from a distribution do not have to be explicitly spent on the expenses for which an exception is being claimed, but the distribution and the expenses must be in the same tax year.
      • Do not ask the taxpayer on what they spent the distribution. Rather, say ‚ÄúIf you had certain types of expenses during the year, we could reduce or eliminate the early withdrawal penalty. Let‚Äôs review your expenses.‚Äù
  • If there is an exception to the additional tax:
    • As discussed in the section immediately above, you should have already viewed the Form 1099-R. Distribution Penalty page, and the second box should have been checked. 
    • Go to Part I of the Form 5329 page [Federal > Other Taxes > Tax on Early Distribution].
    • If a MFJ return, select whether you are completing this page for the taxpayer or the spouse.
    • Enter the amount that is NOT subject to that penalty in the correct box, either SIMPLE [if code S in box 7 of the Form 1099-R] or non-SIMPLE [if code 1]. 
      • Warning:  TaxSlayer does no error checking on the amount entered. Make sure it is correct.
    • Select the reason for exemption in the drop-down box.
    • Click ‚ÄúContinue‚Äù. 
    • On the ‚ÄúOther Taxes‚Äù menu page, click ‚ÄúPrint‚Äù to create a PDF of the Form 5329, to confirm that line 4 of Part I is the expected amount (zero if all distributions subject to penalty have an exception).
    • Note for CA returns: Exception information entered in Part I of Form 5329 automatically carries to CA Form 5805P. 

1099-R Form ‚Äì Transfer or Rollover 

  • A ‚Äútransfer‚Äù is a DIRECT transfer of funds or securities from one qualified account to another. No money passes through the taxpayer‚Äôs hands. (This is also called a ‚Äúdirect rollover‚Äù.) Box 7 has code G or code H.
    • Taxpayers can do this as often as they wish, without penalty.
    • For code G, if there is an amount in box 2a of the paper Form 1099-R, this may be a wholly or partially taxable rollover from a qualified plan to a Roth account. If the taxpayer confirms that the destination for the transfer was a Roth, enter the taxable amount shown in box 2a of the paper form. If the taxpayer is certain that the destination was a traditional IRA or a non-Roth account in another qualified retirement plan, then box 2a in TaxSlayer should be changed to zero.
    • Otherwise, for a code G or code H transfer, box 2a on the paper Form 1099-R should be blank (or zero); in this case, enter zero in box 2a in TaxSlayer.
    • Then in the ‚ÄúRollover or Disability‚Äù section of the Form 1099-R page, check the box ‚ÄúCheck here if all/part of the distribution was rolled over, and enter the rollover amount‚Äù. Then, in the dollar amount box immediately below, enter the box 1 amount as the ‚Äúrollover amount‚Äù (this step shouldn‚Äôt really be required, but TaxSlayer adjusts MAGI used for any sales tax deduction by this amount, for example).
  • A ‚Äúrollover‚Äù is a distribution from a qualified account to the taxpayer, followed by a deposit of some or all of the distribution in the same or a different qualified account. 
    • The deposit must occur WITHIN 60 DAYS of the distribution. 
    • Because rollovers have the same code in box 7 as regular distributions ‚Äì specifically, code 1 or code 7 ‚Äì the only way to know about a rollover is when the taxpayer identifies one.
    • A taxpayer is normally limited to one rollover from an IRA in any 12-month period, regardless of how many accounts there are. 
    • Rollovers are handled as follows:
      • Box 2a should be changed to the amount that was retained by the taxpayer, if any.
        • ‚ÄúRetained‚Äù includes money withheld as taxes. For example, if the taxpayer withdrew $5,000, had $1,000 withheld in taxes, and deposited $4,000 in a different qualified retirement account (not a Roth account), then box 2a should be $1,000.
        • If a taxpayer received a qualified retirement distribution with income tax withheld and then decided to roll over the entire distribution (getting money from another source), then box 2a would be zero.
      • In the ‚ÄúRollover or Disability‚Äù section of Form 1099-R, check the box ‚ÄúCheck here if all/part of the distribution was rolled over, and enter the rollover amount‚Äù. Then in the dollar amount line immediately below, enter the amount that the taxpayer deposited in a retirement account. (This amount plus the amount in box 2a must equal the box 1 amount.) 
      • A taxpayer is not allowed to rollover an inherited IRA unless the IRA was inherited from a spouse. 
  • Revenue Procedure 2016-47 explains a self-certification procedure to help recipients of retirement plan distributions who inadvertently miss the 60-day window for properly rolling these amounts into another retirement account. Eligible taxpayers who meet one or more of 11 circumstances can qualify for a waiver of the 60-day time limit to avoid possible taxes and penalties on early distributions (listed in section 3.02 of the IRS procedure). The IRS document ‚ÄúRetirement Plans FAQs relating to Waivers of the 60-Day Rollover Requirement‚Äù has details.

1099-R Form ‚Äì Qualified Charitable Distribution 

  • The taxpayer may have made one or more Qualified Charitable Distributions (QCD). 
    • This involves the taxpayer directing a payment to go directly to a charity. Such a payment (distribution) is nontaxable, but the taxpayer doesn‚Äôt get a charitable deduction.
      • The taxpayer should have written documentation, should the IRS request more information.
    • The taxpayer must have been at least 70¬Ω when the donation was made; the maximum 2025 donation is $108,000.  The 2024 maximum was $105,000, and $100,000 for any earlier year.
  • New for 2025:  Box 7 may have code Y, in addition to code 4, 7, or K (K is out-of-scope).
  • In box 2a, enter the total amount that the taxpayer received from the retirement account ‚Äì the amount that was not charitable gifts. (This amount is usually zero if all the distribution went to charities.)
    • However, QCDs are taxable to the extent that the taxpayer made deductible contributions to a traditional IRA after age 70¬Ω. For example, if payments to charities totaled $10,000 (box 1), but the taxpayer, age 71, had earned income and contributed $2,000 to a traditional IRA as a deduction (reduction of income), then box 2a in TaxSlayer should be $2,000. (The $2,000 can be used on Schedule A if itemizing.)
      • Warning: The box 2a taxable amount can be affected by deductible IRA contributions made in a prior tax year ‚Äì see NTTC 4012 page D-64.
  • Enter the rest of the 1099-R information as normal; then click the ‚Äúoptions‚Äù link below box 2a.
    • Select the QCD exception.
    • Note: The 1099-R Nontaxable Income page does not code or mark any specific 1099-R as being a QCD distribution; rather, it sends information to the IRS that one or more 1099-Rs involved a QCD.
